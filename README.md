# GitHub User Finder

Автор: Лобанова Софья

## Описание
Приложение позволяет искать пользователей GitHub по имени, просматривать результаты и добавлять понравившихся пользователей в избранное. Избранные пользователи сохраняются в JSON-файл.

## Как использовать API
Введите имя пользователя в поле поиска и нажмите кнопку "Поиск". Если пользователь найден, его данные появятся в списке. Можно добавить его в избранное.

## Примеры использования
- Ввод: `octocat` → отображение информации о пользователе
- Добавление в избранное: выбрать пользователя и нажать "Добавить в избранное"

## Установка и запуск
1. Склонируйте репозиторий
2. Установите необходимые библиотеки (`requests`, `tkinter` — обычно идет в комплекте)
3. Запустите `main.py`
